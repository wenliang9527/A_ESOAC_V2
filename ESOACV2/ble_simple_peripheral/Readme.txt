--------------------------------------------------------------------------------
1. 项目概述 (Project Overview)
--------------------------------------------------------------------------------

本项目是基于 Freqchip FR801x 系列芯片的 BLE (Bluetooth Low Energy) 外设应
用，专为智能空调节能控制系统 (ESOAC - Energy Saving Of Air Conditioners) 设计。

项目集成了 BLE 通信、红外遥控、4G MQTT 通信、ADC 数据采集等功能，实现通过手机
APP 或云端平台对空调设备的远程控制和状态监测。

主要应用场景：
- 智能空调遥控器
- 空调能耗监测设备
- 物联网空调控制器

--------------------------------------------------------------------------------
2. 功能特性 (Features)
--------------------------------------------------------------------------------

2.1 BLE 蓝牙功能
    - 支持 BLE 5.0 协议栈
    - 自定义 GATT 服务 (ESAIR Service, UUID: 0xF002FE00)
    - 支持设备名称动态配置与持久化存储
    - 支持 OTA (Over-The-Air) 无线固件升级
    - 广播间隔可配置，默认 600ms
    - 支持配对绑定管理

2.2 红外遥控功能
    - 支持红外信号学习功能
    - 支持红外信号发射
    - 支持空调控制码库管理
    - 支持电源、模式、温度、风速控制

2.3 4G MQTT 通信
    - 支持 ML307A 4G 模组
    - MQTT 协议连接云端
    - 支持服务器地址、端口、认证信息配置
    - 断线自动重连机制

2.4 数据采集
    - ADC 采集功率和温度数据
    - 定时上报设备状态
    - 心跳包机制

2.5 协议支持
    - 自定义 ESOAC 通信协议
    - 支持 BLE、MQTT、UART 多通道统一协议解析
    - 帧格式：帧头(5A7A) + 长度 + 命令 + 数据 + 校验

--------------------------------------------------------------------------------
3. 硬件平台 (Hardware Platform)
--------------------------------------------------------------------------------

主控芯片：Freqchip FR801x 系列 (ARM Cortex-M3)
- FR8016HA / FR8012HA (4MB Flash)
- FR8016HD / FR8018HD (8MB Flash)  
- FR8012HB (2MB Flash)

外设模块：
- 红外发射/接收模块
- ML307A 4G 通信模组 (UART0)
- 调试串口 (UART1)
- SPI Flash (W25Q 系列)
- ADC 采样电路 (功率、温度)

4.2 任务架构

- 系统任务：BLE 协议栈任务
- 应用任务 (app_task)：处理 BLE/MQTT 事件、定时上报
- 用户任务 (user_task)：按键等用户交互事件
- 红外任务：红外信号发送与接收

4.3 初始化流程

1. user_entry_before_ble_init()    # PMU、Flash、UART 初始化
2. BLE 协议栈初始化
3. user_entry_after_ble_init()     # 外设初始化、GATT 服务注册、MQTT 启动

6. 配置说明 (Configuration)
--------------------------------------------------------------------------------

6.1 Flash 配置 (flash_usage_config.h)

根据芯片型号选择：
    #define FOR_4M_FLASH    // FR8016HA/FR8012HA (默认)
    //#define FOR_8M_FLASH  // FR8016HD/FR8018HD
    //#define FOR_2M_FLASH  // FR8012HB

Flash 保护：
    #define FLASH_PROTECT   // 启用 Flash 写保护

6.2 BLE 配置

设备名称：
    - 默认格式：ESOAC + MAC地址后12位
    - 最大长度：20 字符
    - 支持动态修改并保存到 Flash

广播参数：
    - 广播间隔：600ms (可调)
    - 广播通道：全通道 (37/38/39)
    - 发射功率：0dBm

6.3 MQTT 配置

配置项：
    - 服务器地址 (IP/域名)
    - 服务器端口 (默认 1883)
    - 客户端 ID
    - 用户名/密码
    - 订阅/发布主题

存储：配置保存到 SPI Flash，掉电不丢失

7. 使用方法 (Usage)
--------------------------------------------------------------------------------

7.1 编译与烧录

Keil 方式：
    1. 打开 keil/ble_simple_peripheral.uvprojx
    2. 选择目标设备型号
    3. 编译 (F7)
    4. 使用 J-Link 或串口下载

GCC 方式：
    cd gcc
    make clean
    make

7.2 首次启动

    1. 设备上电后自动开始 BLE 广播
    2. 使用手机 APP 搜索 "ESOACxxxx" 设备
    3. 连接后即可进行控制

7.3 红外学习流程

    1. 发送 CMD_IR_LEARN_START 命令，指定按键编号
    2. 对准原始遥控器按下目标按键
    3. 设备自动学习并保存红外码
    4. 发送 CMD_IR_SAVE_KEYS 保存到 Flash

7.4 MQTT 连接

    1. 通过 BLE 配置 MQTT 服务器参数
    2. 设备自动尝试连接 MQTT 服务器
    3. 连接成功后可通过云端控制设备

8. 通信协议 (Communication Protocol)
--------------------------------------------------------------------------------

8.1 帧格式

    +--------+--------+--------+--------+--------+--------+--------+-----+
    | 帧头1  | 帧头2  | 数据长度| 数据标记| 命令低 | 命令高 | 数据   | 校验 |
    | (0x5A) | (0x7A) | (N+3)  |        | 字节   | 字节   | (N字节)|      |
    +--------+--------+--------+--------+--------+--------+--------+-----+

8.2 主要命令

    CMD_HEARTBEAT       (0x0001)    - 心跳包
    CMD_DEVICE_INFO     (0x0002)    - 获取设备信息
    CMD_SET_POWER       (0x0010)    - 设置电源开关
    CMD_SET_MODE        (0x0011)    - 设置运行模式
    CMD_SET_TEMP        (0x0012)    - 设置温度
    CMD_SET_WIND        (0x0013)    - 设置风速
    CMD_GET_STATUS      (0x0020)    - 获取设备状态
    CMD_GET_POWER       (0x0021)    - 获取功率数据
    CMD_GET_TEMP        (0x0022)    - 获取温度数据
    CMD_IR_LEARN_START  (0x0030)    - 开始学习红外
    CMD_IR_LEARN_STOP   (0x0031)    - 停止学习红外
    CMD_IR_SEND         (0x0032)    - 发送红外信号
    CMD_SET_BLE_NAME    (0x0040)    - 设置 BLE 名称
    CMD_SET_MQTT_CONFIG (0x0050)    - 设置 MQTT 配置

8.3 数据标记

    DATA_MARK_COMMAND   (0x00)      - 命令帧
    DATA_MARK_RESPONSE  (0x01)      - 响应帧
    DATA_MARK_NOTIFY    (0x02)      - 主动上报

9. 注意事项 (Important Notes)
--------------------------------------------------------------------------------

9.1 电源管理
    - 系统默认工作在 Buck 模式
    - 支持睡眠模式以降低功耗
    - 睡眠时 UART1 会输出 's'，唤醒输出 'w'

9.2 Flash 操作
    - Flash 擦写有寿命限制，避免频繁写入
    - 启用 FLASH_PROTECT 防止意外擦除
    - 配置修改时先写入 RAM，确认后再保存到 Flash

9.3 BLE 连接
    - 连接后 4 秒自动请求更新连接参数
    - 断开后自动恢复广播
    - 最多支持 1 个连接

9.4 红外控制
    - 发射红外时系统不能进入睡眠
    - 红外学习时需要屏蔽其他红外干扰
    - 学习数据保存前仅存储在 RAM

9.5 MQTT 连接
    - 首次连接可能需要较长时间 (10-30秒)
    - 网络异常时自动重连，最多重试 10 次
    - 重连间隔：30秒，超限后延长至 120秒

9.6 调试信息
    - 调试串口 UART1，波特率 115200
    - 使用 co_printf 输出日志
    - 关键事件会输出到调试串口
