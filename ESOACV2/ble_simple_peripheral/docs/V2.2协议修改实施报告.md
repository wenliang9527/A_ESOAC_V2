## 二、详细修改内容

### 2.1 frusart.c - ESOACSum函数更新

**修改位置**: frusart.c:19-27

**修改前**:
```c
/* 协议帧校验和：从第 3 字节累加到校验字节前 */
uint8_t ESOACSum(TCommDataPacket buf)
{
  uint8_t i;
  uint8_t ByteSum = 0;
  for (i = 3; i < buf.FramLen - 1; i++) {
    ByteSum += buf.FrameBuf[i];
  }
  return ByteSum;
}
```

**修改后**:
```c
/* 协议帧校验和计算（V2.2版本）
 * 计算范围：帧头(2B) + data_length(1B) + data_mark(1B) + command(2B) + dev_addr(2B) + Data(N)
 * V2.2变更：新增dev_addr字段(2字节)，data_length = 5 + N
 */
uint8_t ESOACSum(TCommDataPacket buf)
{
    uint8_t ByteSum = 0;
    uint16_t i;

    /* 校验和计算范围：从帧头(偏移0)到校验字节前(偏移FramLen-1) */
    for (i = 0; i < buf.FramLen - 1; i++) {
        ByteSum += buf.FrameBuf[i];
    }

    return ByteSum;
}
```

**修改说明**:
- 计算范围从偏移3改为偏移0，确保包含完整的帧头(0x5A 0x7A)
- 使用uint16_t类型的循环变量i，支持更大的帧长度
- 添加V2.2协议注释，说明包含dev_addr字段
- 保持与protocol_calc_checksum()一致的计算逻辑

### 2.2 protocol.c - MQTT配置命令完善

**修改位置**: protocol.c:673-687 (CMD_SET_MQTT_CONFIG), 689-691 (CMD_GET_MQTT_CONFIG)

**修改前**:
```c
case CMD_SET_MQTT_CONFIG:
    /* ... existing MQTT config handling ... */
    protocol_send_response(CMD_SET_MQTT_CONFIG, STATUS_SUCCESS, source);
    break;

case CMD_GET_MQTT_CONFIG:
    /* ... existing MQTT config handling ... */
    protocol_send_response(CMD_GET_MQTT_CONFIG, STATUS_SUCCESS, source);
    break;
```

**修改后**:

#### CMD_SET_MQTT_CONFIG (0x0503)
实现了完整的TLV解析逻辑：

1. **TLV格式解析**:
   - 格式: [type(1B)][len(1B)][value(len)]...
   - type定义: 0=server_addr, 1=server_port, 2=client_id, 3=username, 4=password, 5=subscribe_topic, 6=publish_topic

2. **参数验证**:
   - 检查payload_len >= 2
   - 验证每个TLV的长度不越界
   - 确保至少有一个有效字段

3. **字段更新**:
   - 支持部分更新（只修改指定字段）
   - 使用临时缓冲区避免中间状态污染
   - 合并到g_mqtt_config结构体

4. **持久化**:
   - 调用mqtt_config_save()保存到Flash
   - 发送APP_EVT_MQTT_CONFIG_UPDATED事件通知重连

5. **错误处理**:
   - STATUS_ERROR_PARAM: 参数错误或解析错误
   - STATUS_ERROR_STORAGE: Flash保存失败
   - STATUS_SUCCESS: 更新成功

#### CMD_GET_MQTT_CONFIG (0x0504)
实现了TLV响应构建：

1. **TLV构建**:
   - 遍历7个配置字段
   - 每个字段编码为[type][len][value]
   - 自动计算长度并填充

2. **响应发送**:
   - 使用protocol_build_and_send()发送响应帧
   - 包含当前所有MQTT配置参数
