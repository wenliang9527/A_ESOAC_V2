## 二、V2.2协议关键变更点

| 变更项 | V1版本 | V2.2版本 | 影响 |
|--------|--------|----------|------|
| data_length | = 3 + N | = 5 + N | 帧结构变化 |
| dev_addr字段 | 无 | 2字节，小端 | 新增设备寻址 |
| 最大业务数据 | 252字节 | 250字节 | 容量减少 |
| 校验和范围 | 不含dev_addr | 含dev_addr | 校验计算变化 |
| 广播地址 | 无定义 | 0xFFFF | 新增广播功能 |
| 地址错误响应 | 无 | STATUS_ERROR_ADDR(0x06) | 新增错误码 |
| 设置/获取短地址命令 | 无 | 0x0505/0x0506 | 新增命令 |
