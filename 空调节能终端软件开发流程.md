### 3.1 GPIO驱动开发

**目标**: 配置GPIO用于红外发送/学习、LED指示、按键检测等

**步骤1: 查看SDK GPIO驱动**
```

```

**步骤2: 创建GPIO配置文件**
```
在 code/ 目录创建文件: gpio_config.c 和 gpio_config.h
```

**gpio_config.h**
```c
#ifndef __GPIO_CONFIG_H__
#define __GPIO_CONFIG_H__

#include "driver_gpio.h"

// 红外模块
#define IR_SEND_PIN          GPIO_PIN_PC5     // 红外发送管脚
#define IR_LEARN_INT_PIN     GPIO_PIN_PC6     // 红外学习中断
#define IR_LEARN_EN_PIN      GPIO_PIN_PC7     // 红外学习使能

// LED指示
#define LED_RED_PIN          GPIO_PIN_PA0     // 红色LED
#define LED_GREEN_PIN        GPIO_PIN_PA1     // 绿色LED
#define LED_BLUE_PIN         GPIO_PIN_PA2     // 蓝色LED

// 按键
#define KEY_ENTER_PIN        GPIO_PIN_PA3     // 确认键
#define KEY_MODE_PIN         GPIO_PIN_PA4     // 模式键

// ADC采集
#define ADC_NTC_PIN          GPIO_PIN_PB0     // NTC温度ADC
#define ADC_CURRENT_PIN      GPIO_PIN_PB1     // 电流互感器ADC

// 函数声明
void gpio_init(void);
void gpio_set_led(uint8_t led, uint8_t state);
uint8_t gpio_get_key(uint8_t key);

#endif
```

**gpio_config.c**
```c
#include "gpio_config.h"
#include "driver_system.h"
#include "co_printf.h"

// 初始化GPIO
void gpio_init(void)
{
    // 配置红外发送管脚 (PWM输出)
    gpio_set_direction(IR_SEND_PIN, GPIO_OUTPUT);
    gpio_write(IR_SEND_PIN, 0);

    // 配置红外学习中断 (输入)
    gpio_set_direction(IR_LEARN_INT_PIN, GPIO_INPUT);
    gpio_set_pull(IR_LEARN_INT_PIN, GPIO_PULL_UP);

    // 配置红外学习使能 (输出)
    gpio_set_direction(IR_LEARN_EN_PIN, GPIO_OUTPUT);
    gpio_write(IR_LEARN_EN_PIN, 0);

    // 配置LED (输出)
    gpio_set_direction(LED_RED_PIN, GPIO_OUTPUT);
    gpio_set_direction(LED_GREEN_PIN, GPIO_OUTPUT);
    gpio_set_direction(LED_BLUE_PIN, GPIO_OUTPUT);
    gpio_write(LED_RED_PIN, 0);
    gpio_write(LED_GREEN_PIN, 0);
    gpio_write(LED_BLUE_PIN, 0);

    // 配置按键 (输入, 上拉)
    gpio_set_direction(KEY_ENTER_PIN, GPIO_INPUT);
    gpio_set_pull(KEY_ENTER_PIN, GPIO_PULL_UP);
    gpio_set_direction(KEY_MODE_PIN, GPIO_INPUT);
    gpio_set_pull(KEY_MODE_PIN, GPIO_PULL_UP);

    co_printf("[GPIO] Init complete\r\n");
}

// 设置LED状态
void gpio_set_led(uint8_t led, uint8_t state)
{
    switch(led) {
        case 0: // 红
            gpio_write(LED_RED_PIN, state);
            break;
        case 1: // 绿
            gpio_write(LED_GREEN_PIN, state);
            break;
        case 2: // 蓝
            gpio_write(LED_BLUE_PIN, state);
            break;
    }
}

// 读取按键状态
uint8_t gpio_get_key(uint8_t key)
{
    uint8_t value;
    switch(key) {
        case 0: // ENTER
            value = gpio_read(KEY_ENTER_PIN);
            break;
        case 1: // MODE
            value = gpio_read(KEY_MODE_PIN);
            break;
        default:
            return 0;
    }
    // 低电平有效, 返回1表示按下
    return (value == 0) ? 1 : 0;
}
```

```c
#include "adc_config.h"
#include "co_printf.h"

// ADC初始化
void adc_init(void)
{
    // 使能ADC时钟
    system_periph_clk_enable(PERIPH_CLK_ADC);

    // 配置ADC参数
    adc_config_t adc_cfg;
    adc_cfg.sample_rate = ADC_SAMPLE_FREQ;
    adc_cfg.resolution = ADC_RESOLUTION;
    adc_cfg.mode = ADC_MODE_SINGLE;

    // 初始化ADC
    if (adc_init_module(&adc_cfg) == 0) {
        co_printf("[ADC] Init complete\r\n");
    } else {
        co_printf("[ADC] Init failed!\r\n");
    }
}

// 读取NTC ADC值
uint16_t adc_read_ntc(void)
{
    return adc_read_channel(ADC_CHANNEL_NTC);
}

// 读取电流ADC值
uint16_t adc_read_current(void)
{
    return adc_read_channel(ADC_CHANNEL_CURRENT);
```

## 四、功能模块开发

### 4.1 4G AT指令通讯模块

**目标**: 通过AT指令控制ML307A 4G模块,实现MQTT协议通讯

**步骤1: 创建4G模块文件**
```
在 code/ 目录创建: ml307a.c 和 ml307a.h
```

**ml307a.h**
```c
#ifndef __ML307A_H__
#define __ML307A_H__

#include <stdint.h>
#include <stdbool.h>

// 4G模块状态枚举
typedef enum {
    ML307A_STATE_IDLE = 0,
    ML307A_STATE_INITIALIZING,
    ML307A_STATE_NETWORK_ATTACHING,
    ML307A_STATE_CONNECTED,
    ML307A_STATE_MQTT_CONNECTING,
    ML307A_STATE_MQTT_CONNECTED,
    ML307A_STATE_DISCONNECTED,
    ML307A_STATE_ERROR
} ml307a_state_e;

// MQTT配置结构
typedef struct {
    char server[128];        // MQTT服务器地址
    uint16_t port;          // MQTT服务器端口
    char client_id[64];     // 客户端ID
    char username[64];      // 用户名
    char password[64];      // 密码
} mqtt_config_t;

// 函数声明
void ml307a_init(void);
bool ml307a_check_ready(void);
bool ml307a_send_at_cmd(const char *cmd, const char *expected_rsp, uint32_t timeout);
bool ml307a_network_attach(void);
bool ml307a_mqtt_connect(mqtt_config_t *config);
bool ml307a_mqtt_publish(const char *topic, const char *payload, uint8_t qos);
bool ml307a_mqtt_subscribe(const char *topic, uint8_t qos);
ml307a_state_e ml307a_get_state(void);
void ml307a_process_response(char *resp, uint16_t len);

#endif
```

**ml307a.c (简化版)**
```c
#include "ml307a.h"
#include "co_printf.h"
#include <string.h>
#include <stdio.h>

// 4G模块状态
static ml307a_state_e ml307a_state = ML307A_STATE_IDLE;

// AT指令发送 (简化版)
bool ml307a_send_at_cmd(const char *cmd, const char *expected_rsp, uint32_t timeout)
{
    char buffer[256];
    uint32_t start = system_get_tick();

    co_printf("[ML307A] Send AT: %s\r\n", cmd);

    // 发送AT指令到UART1 (连接到ML307A)
    uart_send_string(UART1, cmd);
    uart_send_string(UART1, "\r\n");

    // 等待响应
    while ((system_get_tick() - start) < timeout) {
        if (uart_receive_line(UART1, buffer, sizeof(buffer)) > 0) {
            if (strstr(buffer, expected_rsp) != NULL) {
                return true;
            }
            if (strstr(buffer, "ERROR") != NULL) {
                co_printf("[ML307A] AT error: %s\r\n", buffer);
                return false;
            }
        }
        co_delay_ms(10);
    }
    return false;
}

// 初始化4G模块
void ml307a_init(void)
{
    ml307a_state = ML307A_STATE_INITIALIZING;
    co_printf("[ML307A] Init...\r\n");

    // 配置UART1用于AT指令通信 (波特率115200)
    uart_config_t uart_cfg;
    uart_cfg.baudrate = 115200;
    uart_cfg.data_bits = 8;
    uart_cfg.stop_bits = 1;
    uart_cfg.parity = UART_PARITY_NONE;
    uart_init(UART1, &uart_cfg);

    // 等待模块就绪
    if (ml307a_send_at_cmd("AT", "OK", 1000)) {
        co_printf("[ML307A] Module ready\r\n");
        ml307a_state = ML307A_STATE_IDLE;
    } else {
        co_printf("[ML307A] Module not ready\r\n");
        ml307a_state = ML307A_STATE_ERROR;
    }
}

// 网络附着
bool ml307a_network_attach(void)
{
    ml307a_state = ML307A_STATE_NETWORK_ATTACHING;

    // 检查SIM卡
    if (!ml307a_send_at_cmd("AT+CPIN?", "READY", 5000)) {
        co_printf("[ML307A] SIM card error\r\n");
        return false;
    }

    // 设置APN
    ml307a_send_at_cmd("AT+CGDCONT=1,\"IP\",\"CMNET\"", "OK", 1000);

    // 激活网络
    if (!ml307a_send_at_cmd("AT+CGACT=1,1", "OK", 30000)) {
        co_printf("[ML307A] Network activate failed\r\n");
        return false;
    }

    // 查询附着状态
    if (ml307a_send_at_cmd("AT+CGREG?", ",1", 10000)) {
        co_printf("[ML307A] Network attached\r\n");
        ml307a_state = ML307A_STATE_CONNECTED;
        return true;
    }

    ml307a_state = ML307A_STATE_ERROR;
    return false;
}

// MQTT连接
bool ml307a_mqtt_connect(mqtt_config_t *config)
{
    if (config == NULL) return false;
    ml307a_state = ML307A_STATE_MQTT_CONNECTING;

    char cmd[256];

    // 打开MQTT客户端
    snprintf(cmd, sizeof(cmd), "AT+QMTOPEN=0,\"%s\",%d",
             config->server, config->port);
    if (!ml307a_send_at_cmd(cmd, "OK", 10000)) {
        return false;
    }

    // 连接MQTT服务器
    snprintf(cmd, sizeof(cmd), "AT+QMTCONN=0,\"%s\",\"%s\",\"%s\"",
             config->client_id, config->username, config->password);

    if (ml307a_send_at_cmd(cmd, "0,0,0", 10000)) {
        co_printf("[ML307A] MQTT connected\r\n");
        ml307a_state = ML307A_STATE_MQTT_CONNECTED;
        return true;
    }

    ml307a_state = ML307A_STATE_ERROR;
    return false;
}

// MQTT发布
bool ml307a_mqtt_publish(const char *topic, const char *payload, uint8_t qos)
{
```

**protocol.c**
```c
#include "protocol.h"
#include <string.h>

// 计算校验和
static uint8_t protocol_calc_checksum(uint8_t *data, uint16_t len)
{
    uint8_t checksum = 0;
    for (uint16_t i = 0; i < len; i++) {
        checksum += data[i];
    }
    return checksum;
}

// 打包协议帧
uint16_t protocol_pack_frame(uint8_t *out_buf, uint8_t data_mark,
                             uint16_t command, uint8_t *data, uint8_t data_len)
{
    if (out_buf == NULL || data_len > 254) {
        return 0;
    }

    uint16_t index = 0;

    // 帧头
    out_buf[index++] = 0x5A;
    out_buf[index++] = 0x7A;

    // 数据长度 (去掉帧头和校验位)
    out_buf[index++] = 1 + 2 + data_len;  // data_mark + command + data

    // 数据标记
    out_buf[index++] = data_mark;

    // 命令字节
    out_buf[index++] = (command >> 8) & 0xFF;
    out_buf[index++] = command & 0xFF;

    // 数据字节
    if (data != NULL && data_len > 0) {
        memcpy(&out_buf[index], data, data_len);
        index += data_len;
    }

    // 校验和 (从数据长度开始计算)
    uint8_t checksum = protocol_calc_checksum(&out_buf[2], index - 2);
    out_buf[index++] = checksum;

    return index;  // 返回帧总长度
}

// 解析协议帧
bool protocol_parse_frame(uint8_t *in_buf, uint16_t len,
                          uint8_t *data_mark, uint16_t *command,
                          uint8_t *data, uint8_t *data_len)
{
    if (in_buf == NULL || len < 7) {  // 最小帧长度: 2+1+1+2+0+1 = 7
        return false;
    }

    // 检查帧头
    if (in_buf[0] != 0x5A || in_buf[1] != 0x7A) {
        return false;
    }

    uint16_t index = 2;
    uint8_t frame_data_len = in_buf[index++];  // 数据长度

    // 验证帧长度
    if (len < 7 + frame_data_len) {
        return false;
    }

    // 数据标记
    if (data_mark != NULL) {
        *data_mark = in_buf[index++];
    } else {
        index++;
    }

    // 命令字节
    if (command != NULL) {
        *command = (in_buf[index] << 8) | in_buf[index + 1];
    }
    index += 2;

    // 数据字节
    uint8_t payload_len = frame_data_len - 3;  // data_mark + command
    if (data != NULL && data_len != NULL && payload_len > 0) {
        *data_len = payload_len;
        memcpy(data, &in_buf[index], payload_len);
    }
    index += payload_len;

    // 校验和验证
    uint8_t checksum = protocol_calc_checksum(&in_buf[2], index - 2);
    if (checksum != in_buf[index]) {
        return false;
    }

    return true;
}
```

```c
#include "ntc_sensor.h"
#include "adc_config.h"
#include "math.h"
#include "co_printf.h"

// Steinhart-Hart方程系数
#define STEINHART_A         0.001129148f
#define STEINHART_B         0.000234125f
#define STEINHART_C         0.0000000876741f

// 滤波缓冲区
static float temp_buffer[TEMP_FILTER_SIZE];
static uint8_t temp_index = 0;
static float calibrate_offset = 0.0f;

// NTC初始化
void ntc_init(void)
{
    // 清空滤波缓冲区
    for (int i = 0; i < TEMP_FILTER_SIZE; i++) {
        temp_buffer[i] = 25.0f;  // 初始化为25°C
    }
    co_printf("[NTC] Init complete\r\n");
}

// 读取温度值 (已滤波)
float ntc_read_temperature(void)
{
    // 读取ADC原始值
    uint16_t adc_value = adc_read_ntc();

    // 转换为电压 (3.3V参考)
    float voltage = (float)adc_value / 4095.0f * 3.3f;

    // 计算NTC电阻
    float r_ntc = NTC_R_FIX * (3.3f / voltage - 1.0f);

    // 使用Steinhart-Hart方程计算温度
    float log_r = logf(r_ntc);
    float temp_kelvin = 1.0f / (STEINHART_A +
                                STEINHART_B * log_r +
                                STEINHART_C * log_r * log_r * log_r);

    // 转换为摄氏度
    float temp_celsius = temp_kelvin - 273.15f;

    // 应用校准偏移
    temp_celsius += calibrate_offset;

    // 范围检查
    if (temp_celsius < TEMP_MIN) temp_celsius = TEMP_MIN;
    if (temp_celsius > TEMP_MAX) temp_celsius = TEMP_MAX;

    // 滑动平均滤波
    temp_buffer[temp_index] = temp_celsius;
    temp_index = (temp_index + 1) % TEMP_FILTER_SIZE;

    float sum = 0.0f;
    for (int i = 0; i < TEMP_FILTER_SIZE; i++) {
        sum += temp_buffer[i];
    }

    return sum / TEMP_FILTER_SIZE;
}

// 温度校准
void ntc_calibrate(float offset)
{
    calibrate_offset = offset;
    co_printf("[NTC] Calibrate offset: %.2f\r\n", offset);
}
```

**power_monitor.c**
```c
#include "power_monitor.h"
#include "adc_config.h"
#include "co_printf.h"
#include <math.h>

// 功率数据
static power_data_t power_data = {0};
static float energy_accumulator = 0.0f;
static uint32_t last_update_time = 0;

// 功率监测初始化
void power_monitor_init(void)
{
    // 清零数据
    memset(&power_data, 0, sizeof(power_data_t));
    energy_accumulator = 0.0f;
    last_update_time = 0;

    co_printf("[POWER] Init complete\r\n");
}

// 读取电流值 (A)
float power_read_current(void)
{
    // 读取ADC值 (去掉直流偏置)
    uint16_t adc_raw = adc_read_current();
    int16_t adc_signed = (int16_t)adc_raw - 2048;  // 中心点为2048

    // 转换为电压 (3.3V参考)
    float voltage = (float)adc_signed / 4095.0f * 3.3f;

    // 计算电流互感器二次侧电流
    float i_ct = voltage / CT_R_SAMPLE;

    // 计算一次侧电流 (负载电流)
    float i_load = i_ct * CT_RATIO;

    // 取绝对值 (交流)
    return fabsf(i_load);
}

// 计算功率 (W)
float power_calculate_watts(void)
{
    float current = power_read_current();
    float power = current * V_MAINS * POWER_FACTOR;

    return power;
}

// 检测空调运行状态
ac_state_e power_detect_ac_state(void)
{
    float power = power_calculate_watts();

    if (power < POWER_THRESHOLD_OFF) {
        return AC_STATE_OFF;
    } else if (power < POWER_THRESHOLD_STANDBY) {
        return AC_STATE_STANDBY;
    } else if (power < POWER_THRESHOLD_RUNNING) {
        return AC_STATE_STARTUP;
    } else {
        return AC_STATE_RUNNING;
    }
}

// 获取实时功率数据
power_data_t power_get_realtime_data(void)
{
    // 更新电流和功率
    power_data.current = power_read_current();
    power_data.power = power_calculate_watts();
    power_data.voltage = V_MAINS;

    // 检测状态
    power_data.state = power_detect_ac_state();

    // 更新峰值功率
    if (power_data.power > power_data.peak_power) {
        power_data.peak_power = power_data.power;
    }

    // 累计电量 (假设每5秒调用一次)
    uint32_t current_time = system_get_tick();  // 获取系统时间
    if (last_update_time > 0) {
        uint32_t delta_time = (current_time - last_update_time) / 1000;  // 秒
        float energy_delta = power_data.power * delta_time / 3600.0f;     // Wh
        energy_accumulator += energy_delta;
    }
    last_update_time = current_time;

    power_data.energy_wh = energy_accumulator;
    power_data.timestamp = current_time;

    return power_data;
}

// 重置电量累计
void power_reset_energy(void)
{
    energy_accumulator = 0.0f;
    power_data.peak_power = 0.0f;
    last_update_time = 0;

    co_printf("[POWER] Energy reset\r\n");
}
```

```c
                ntc_calibrate(temp_offset);
                // power_set_pf(power_pf);

                uint8_t rsp[] = {0};  // 成功
                airsync_send_rsp(conidx, cmd_id, rsp, 1);
            }
            break;
        }

        case CMD_ID_CONTROL: {
            // 空调控制命令 (格式: [按键ID])
            if (payload_len >= 1) {
                ir_key_e key = (ir_key_e)payload[0];
                ir_send_key(key);

                uint8_t rsp[] = {0};  // 成功
                airsync_send_rsp(conidx, cmd_id, rsp, 1);
            }
            break;
        }

        case CMD_ID_LEARN_IR: {
            // 学习红外命令 (格式: [按键ID])
            if (payload_len >= 1) {
                ir_key_e key = (ir_key_e)payload[0];
                ir_start_learn();

                uint8_t rsp[] = {0};  // 成功
                airsync_send_rsp(conidx, cmd_id, rsp, 1);
            }
            break;
        }

        case CMD_ID_QUERY_STATUS: {
            // 查询状态命令
            power_data_t power = power_get_realtime_data();
            float temp = ntc_read_temperature();

            // 响应格式: [温度][电流][功率][状态][电量]
            uint8_t rsp[20];
            memcpy(&rsp[0], &temp, 4);
            memcpy(&rsp[4], &power.current, 4);
            memcpy(&rsp[8], &power.power, 4);
            rsp[12] = (uint8_t)power.state;
            memcpy(&rsp[13], &power.energy_wh, 4);

            airsync_send_rsp(conidx, cmd_id, rsp, 17);
            break;
        }

        case CMD_ID_RESET: {
            // 重置命令
            power_reset_energy();

            uint8_t rsp[] = {0};  // 成功
            airsync_send_rsp(conidx, cmd_id, rsp, 1);
            break;
        }

        default:
            co_printf("[AirSync] Unknown cmd: 0x%02X\r\n", cmd_id);
            break;
    }
}
```

```c
// 处理MQTT接收到的消息
void mqtt_handle_message(char *topic, uint8_t *data, uint16_t len)
{
    if (data == NULL || len < 7) {
        return;
    }

    co_printf("[MQTT] Received: %s, len: %d\r\n", topic, len);

    // 解析协议帧
    uint8_t data_mark, cmd_data[254];
    uint16_t command;
    uint8_t data_len;

    if (!protocol_parse_frame(data, len, &data_mark, &command, cmd_data, &data_len)) {
        co_printf("[MQTT] Protocol parse failed\r\n");
        return;
    }

    co_printf("[MQTT] Command: 0x%04X\r\n", command);

    // 处理命令
    switch (command) {
        case CMD_TEST: {
            // 数据传输测试
            uint8_t rsp_buf[64];
            uint16_t rsp_len = protocol_pack_frame(rsp_buf, DATA_MARK_TEST, CMD_TEST, NULL, 0);
            // ml307a_mqtt_publish("esoad/response", (char*)rsp_buf, rsp_len, 0);
            break;
        }

        case CMD_TIME_SYNC: {
            // 时间校准 (格式: 年月日时分秒周)
            if (data_len >= 7) {
                co_printf("[MQTT] Time sync: %02X%02X%02X %02X%02X%02X\r\n",
                       cmd_data[0], cmd_data[1], cmd_data[2],
                       cmd_data[3], cmd_data[4], cmd_data[5]);
                // TODO: 设置系统时间
            }
            break;
        }

        case CMD_KEY_LEARN: {
            // 按键学习 (格式: [按键ID])
            if (data_len >= 1) {
                uint8_t key_id = cmd_data[0];
                ir_start_learn(key_id);
                co_printf("[MQTT] Start learning key: %d\r\n", key_id);
            }
            break;
        }

        case CMD_KEY_SEND: {
            // 按键发送 (格式: [按键ID])
            if (data_len >= 1) {
                uint8_t key_id = cmd_data[0];
                ir_send_key(key_id);
                co_printf("[MQTT] Send key: %d\r\n", key_id);
            }
            break;
        }

        case CMD_KEY_SAVE: {
            // 学习键值保存
            ir_save_all_keys();
            co_printf("[MQTT] Keys saved\r\n");
            break;
        }

        case CMD_KEY_LOAD: {
            // 从储存中读取键值
            ir_load_all_keys();
            co_printf("[MQTT] Keys loaded\r\n");
            break;
        }

        case CMD_AC_POWER: {
            // 空调开关机
            ir_send_key(KEY_POWER);
            break;
        }

        case CMD_AC_TEMP_UP: {
            // 温度+
            ir_send_key(KEY_TEMP_UP);
            break;
        }

        case CMD_AC_TEMP_DOWN: {
            // 温度-
            ir_send_key(KEY_TEMP_DOWN);
            break;

        case CMD_AC_MODE: {
            // 模式切换
            ir_send_key(KEY_MODE);
            break;

        case CMD_AC_FAN: {
            // 风速
            ir_send_key(KEY_FAN);
            break;
        }

        default:
            co_printf("[MQTT] Unknown command: 0x%04X\r\n", command);
            break;
    }
}
```

```c
#include "energy_control.h"
#include "ntc_sensor.h"
#include "power_monitor.h"
#include "ir_control.h"
#include "co_printf.h"

// 节能控制参数
static energy_mode_e current_mode = ENERGY_MODE_AUTO;
static energy_stats_t energy_stats = {0};
static float target_temp_cool = 26.0f;  // 制冷基准温度
static float target_temp_heat = 22.0f;  // 制热基准温度

// 节能控制初始化
void energy_control_init(void)
{
    memset(&energy_stats, 0, sizeof(energy_stats_t));
    co_printf("[ENERGY] Init complete\r\n");
}

// 设置节能模式
void energy_control_set_mode(energy_mode_e mode)
{
    current_mode = mode;
    co_printf("[ENERGY] Mode set to: %d\r\n", mode);
}

// 节能控制主循环 (定时调用)
void energy_control_run(void)
{
    // 读取环境温度
    float temp = ntc_read_temperature();

    // 获取功率数据
    power_data_t power = power_get_realtime_data();

    // 根据模式调整
    float target_temp = target_temp_cool;  // 默认制冷模式
    bool need_adjust = false;

    switch (current_mode) {
        case ENERGY_MODE_AUTO: {
            // 自动模式
            if (temp > 28.0f) {
                target_temp = 24.0f;  // 快速降温
            } else if (temp > 26.0f) {
                target_temp = 26.0f;  // 正常制冷
            } else if (temp < 20.0f) {
                target_temp = 24.0f;  // 快速制热
                target_temp_heat = target_temp;
            } else if (temp < 22.0f) {
                target_temp = 22.0f;  // 正常制热
                target_temp_heat = target_temp;
            } else {
                // 温度适宜,无需调节
                return;
            }
            need_adjust = true;
            break;
        }

        case ENERGY_MODE_ECO: {
            // 节能模式: 提高设定温度/降低设定温度
            if (power.power > 500.0f) {
                if (temp > 25.0f) {
                    target_temp = 28.0f;  // 夏季节能
                } else {
                    target_temp = 20.0f;  // 冬季节能
                }
                need_adjust = true;
            }
            break;
        }

        case ENERGY_MODE_COMFORT: {
            // 舒适模式: 保持恒温
            if (fabsf(temp - target_temp) > 1.0f) {
                need_adjust = true;
            }
            break;
        }

        case ENERGY_MODE_OFF:
        default:
            return;
    }

    // 执行温度调节
    if (need_adjust && power.state == AC_STATE_RUNNING) {
        // 发送温度调节指令
        // (需要根据温度差发送TEMP_UP或TEMP_DOWN)
        co_printf("[ENERGY] Adjust to %.1f°C (current: %.1f°C)\r\n",
                  target_temp, temp);

        // 更新节能统计
        if (energy_stats.energy_baseline > 0) {
            energy_stats.energy_actual += power.power / 60.0f;  // 假设每分钟调用
            energy_stats.energy_saved = energy_stats.energy_baseline - energy_stats.energy_actual;
            if (energy_stats.energy_baseline > 0) {
                energy_stats.saving_rate = energy_stats.energy_saved / energy_stats.energy_baseline * 100.0f;
            }
        }
    }
}

// 获取节能统计
energy_stats_t energy_control_get_stats(void)
{
    return energy_stats;
}
```

### 7.3 OTA升级准备

**1. 分区规划**
```
Flash分区布局:
0x10000000 - 0x10080000: 应用程序 A区 (512KB)
0x10080000 - 0x10100000: 应用程序 B区 (512KB)
0x10100000 - 0x10120000: 配置数据 (128KB)
0x10120000 - 0x10180000: 固件缓存 (384KB)
0x10180000 - 0x10200000: 保留空间 (512KB)
```

**2. OTA流程实现**
```
参考 SDK: components/ble/profiles/ble_ota/

实现步骤:
1. 接收新固件数据包
2. 写入临时分区
3. 校验固件完整性 (CRC32/SHA256)
4. 写入启动标志
5. 重启切换分区
6. 上报升级结果
```

## 八、常见问题处理

### 8.1 编译错误

**问题1: 找不到头文件**
```
错误信息: Fatal Error: Cannot open file "xxx.h"

解决方法:
1. 检查头文件路径是否正确
2. Project → Options → C/C++ → Include Paths
3. 添加缺失的路径:
   ../../components/ble/include
   ../../components/driver/include
   ../../components/modules
```

**问题2: 链接错误**
```
错误信息: L6218E: Undefined symbol xxx

解决方法:
1. 检查是否添加了对应的 .c 文件到工程
2. 检查库文件是否正确链接
3. 检查函数声明是否正确
```

**问题3: Flash溢出**
```
错误信息: Error: L6050U: Size of image exceeds region

解决方法:
1. 优化编译选项 (-O3)
2. 删除未使用的代码
3. 增大Flash分区大小
4. 使用代码压缩
```

### 8.2 运行时问题

**问题1: 程序启动后无响应**
```
可能原因:
1. 硬件连接问题
2. 时钟配置错误
3. 中断未使能

解决方法:
1. 检查电源和晶振
2. 使用Keil调试查看程序执行位置
3. 检查 main() 函数是否执行
4. 添加串口调试输出
```

**问题2: ADC采集值异常**
```
可能原因:
1. ADC通道配置错误
2. 参考电压设置错误
3. 硬件连接问题

解决方法:
1. 验证ADC配置代码
2. 用万用表测量ADC管脚电压
3. 检查分压电阻是否正确
4. 添加滤波电容
```

**问题3: 蓝牙无法连接**
```
可能原因:
1. 广播未开启
2. 设备名称设置错误
3. 服务UUID不匹配

解决方法:
1. 检查是否调用 airsync_start_advertising()
2. 验证广播数据格式
3. 使用 nRF Connect 工具扫描设备
4. 检查服务UUID是否正确: 0xFEE7
```

### 8.3 性能问题

**问题1: 功耗过高**
```
解决方法:
1. 进入低功耗模式
2. 降低采样频率
3. 关闭未使用的外设
4. 优化蓝牙广播间隔
```

**问题2: 响应慢**
```
解决方法:
1. 优化算法复杂度
2. 使用DMA传输
3. 增加缓冲区大小
4. 减少阻塞操作
```

### 8.4 调试技巧

**1. 使用串口日志**

```c
// 添加调试日志
```
